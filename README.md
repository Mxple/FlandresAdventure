# Winter-Blossoms
Winter-themed platformer game for GaSTC category Digital Game Design. Coded in Gamemaker Studio 2.

## Features
- Save/Load
- Ambient lighting
- Custom rendering through shaders
- Adjustable controls
- Polished player and enemy animations
- Three unique palletes
